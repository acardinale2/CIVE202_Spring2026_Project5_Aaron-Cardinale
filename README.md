# Project-5
### This coding project analyzes drinking water access in Sub-Saharan Africa using the JMP dataset. The project displays how access has improved over time, and finds the countries that are most at need for clean water. The Findings Show that Angola and Kenya still have over a 20% gap in their populations drinking water access, and our firm believes the water project should focus funding on those areas.
